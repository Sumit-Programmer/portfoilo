Project 1

LibreOffice Writer – Project Report & Documentation

In LibreOffice Writer, I prepared the complete project file for my Hospital Database and other IT work. I focused on professional formatting so that the document looks like a real report.

  • Created a **cover page** with project title, my name, class and subject.
  • Added **page layout**: margins, line spacing and alignment for clean text.
  • Used **styles** (Heading 1, Heading 2, Normal) for proper structure.
  • Inserted **header & footer** with school name and page numbers.
  • Added **tables** to organise information clearly.
  • Inserted **screenshots** of each step of my Base and Calc work.
  • Exported the final report to **PDF** for submission.
What I learnt

Document Design Skills

  • How to make documents look neat and professional.
  • How to organise long reports using headings and sections.
  • How to combine text, tables and images in one file.

Project 2

LibreOffice Calc – Data & Calculations

In LibreOffice Calc, I created spreadsheets to manage and analyse data. This helped me understand how businesses and organisations use spreadsheets.

  • Created a **marks sheet** / **billing sheet** with rows and columns.
  • Used basic formulas like =SUM(), =AVERAGE().
  • Practiced **cell referencing** (A1, B2, etc.).
  • Applied **sorting and filtering** to view specific data.
  • Created a **chart/graph** to visually show results.
  • Formatted cells (borders, colours, number formats) to improve readability.
What I learnt

Analytical & Practical Skills

  • How to use spreadsheets for calculations instead of manual work.
  • How visual charts can explain data quickly.
  • Basic data analysis concepts used in real offices.

Project 3

LibreOffice Base – Hospital Database Project

My main IT project was creating a Hospital Management Database using LibreOffice Base. This project shows how databases are used to store and manage information.

  • Planned the database structure for a hospital scenario.
  • Created **tables** like:
    • Patients
    • Doctors
    • Appointments
    • Billing / Payments
  • Set **field types** (Text, Number, Date, etc.) and primary keys.
  • Defined **relationships** between tables (e.g., one doctor – many patients).
  • Designed **forms** for easy data entry (e.g., add new patient, new appointment).
  • Created **queries** to search and filter data (e.g., patients by doctor, date).
  • Generated **reports** to print summaries, like daily appointments or total bills.
What I learnt

Real Database Concepts

  • How data is connected using tables and relationships.
  • The difference between documents, spreadsheets and databases.
  • How hospitals and other organisations can manage data digitally.
✔ Highlight for Teacher: I combined all three tools – Writer, Calc and Base – in one connected project: Base for the database, Calc for calculations/charts, and Writer for the final report.
